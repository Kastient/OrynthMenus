# История изменений

[← На главную](README.md)

## 1.0.1 — 2026-07-12

- исправлено последовательное редактирование нескольких кнопок через Shift + клик;
- добавлена защита перехода от повторных Bedrock-пакетов и устаревших форм;
- строки lore теперь разделяются символом `|`, количество строк не ограничено;
- возвращение из настроек в FakeInventories выполняется с безопасной задержкой.

## 1.0.0 — 2026-07-12

### Основные возможности

- Inventory GUI через FakeInventories 1.1.8 и нативные Bedrock Forms;
- автоматическая совместимость с новым и старым API FakeInventories;
- обход конфликта, когда другой плагин встроил старый `me.iwareq.fakeinventories.*` без relocation;
- Bedrock-панель `/omenus settings` и настоящий сундук-редактор без стеклянных заглушек;
- перенос предметов и стаков между инвентарём и меню, автоматические `item_1`, `item_2`, `slot` и `slots`;
- настройки имени, lore, количества, Glow, атрибутов и action через Shift + клик;
- обычная и донатная валюта на уровне каждого товара;
- атомарные `trade`-операции покупки и продажи;
- `open_requirement`, `view_requirement`, `click_requirement`, отрицания, логика «ИЛИ», regex, строковые/числовые сравнения, мир, режим и расстояние;
- история меню и `[back]`;
- безопасные `delay-ticks` и `chance` для визуальных действий;
- атомарный reload, валидация YAML и резервные копии;
- PlaceholderAPI-nukkit и Java API для собственных типов требований.
