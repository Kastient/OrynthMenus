# Быстрый старт

## 🚀 Быстрый старт

На этой странице вы создадите своё первое рабочее меню и откроете его командой.

> 💡 Для примера будет создано простое меню магазина с отображением баланса игрока.

### 1. Откройте основной конфиг

Файл регистрации меню находится здесь:

```
plugins/OrynthMenus/config.yml
```

Добавьте новое меню:

```yaml
gui_menus:
  shop:
    file: shops/main.yml
```

Где:

| Поле   | Значение                                |
| ------ | --------------------------------------- |
| `shop` | Уникальный ID меню                      |
| `file` | Путь к файлу относительно папки `menus` |

### 2. Создайте файл меню

Создайте папку:

```
plugins/OrynthMenus/menus/shops/
```

Внутри создайте файл:

```
main.yml
```

Полный путь:

```
plugins/OrynthMenus/menus/shops/main.yml
```

### 3. Вставьте конфигурацию

```yaml
type: inventory
inventory-type: CHEST
size: 27
title: "&8Магазин"
update_interval: 1

open:
  command: shop
  description: "Открыть магазин"

items:
  balance:
    material: "minecraft:emerald"
    slot: 15
    display_name: "&aВаш баланс"
    lore:
      - "&7Монеты: &e%OrynthMenus_balance%"
    update: true

  close:
    material: "minecraft:barrier"
    slot: 23
    display_name: "&cЗакрыть"
    lore:
      - "&7Нажмите, чтобы закрыть меню"
    action:
      - "[sound] random.click"
      - "[close]"
```

### 4. Перезагрузите конфиги

Выполните:

```
/orynthmenus reload
```

или:

```
/omenus reload
```

### 5. Откройте меню

Используйте команду:

```
/shop
```

Если всё настроено правильно, откроется меню на 27 слотов.

### Как работает этот пример

#### Основные параметры

```yaml
type: inventory
```

Указывает, что используется инвентарное меню.

```yaml
inventory-type: CHEST
```

Определяет тип контейнера.

```yaml
size: 27
```

Задаёт размер меню.

```yaml
title: "&8Магазин"
```

Задаёт заголовок.

#### Команда открытия

```yaml
open:
  command: shop
  description: "Открыть магазин"
```

Создаёт команду:

```
/shop
```

#### Динамический баланс

```yaml
lore:
  - "&7Монеты: &e%OrynthMenus_balance%"
update: true
```

Баланс будет автоматически обновляться, пока меню открыто.

#### Кнопка закрытия

```yaml
action:
  - "[sound] random.click"
  - "[close]"
```

Сначала воспроизводится звук, затем меню закрывается.

### Важные правила

> ⚠️ Отступы в YAML должны состоять из пробелов. Не используйте табуляцию.

Правильно:

```yaml
items:
  balance:
    material: "minecraft:emerald"
```

Неправильно:

```yaml
items:
 balance:
   material: "minecraft:emerald"
```

### Проверка перед запуском

* [ ] Меню зарегистрировано в `config.yml`.
* [ ] Файл существует в папке `menus`.
* [ ] Путь указан без ошибок.
* [ ] В YAML нет табуляции.
* [ ] ID меню уникален.
* [ ] Команда не занята другим плагином.
* [ ] FakeInventories установлен.
* [ ] Конфиги перезагружены.
