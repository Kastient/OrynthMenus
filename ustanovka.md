# Установка

## 📦 Установка

Эта страница поможет правильно установить OrynthMenus и подготовить сервер к созданию первых меню.

> ✅ Перед началом убедитесь, что сервер запускается на актуальной сборке **NukkitMOT** и у вас есть доступ к папке `plugins`.

### Требования

Для работы OrynthMenus понадобятся:

| Компонент             |           Статус | Назначение                          |
| --------------------- | ---------------: | ----------------------------------- |
| NukkitMOT             |      Обязательно | Серверное ядро                      |
| OrynthMenus           |      Обязательно | Основной плагин меню                |
| FakeInventories       |      Обязательно | Отображение инвентарных интерфейсов |
| EconomyAPI            | По необходимости | Деньги, покупки и проверки баланса  |
| PlaceholderAPI-nukkit |    Необязательно | Внешние плейсхолдеры                |

### 1. Установка OrynthMenus

Поместите файл:

```
OrynthMenus-1.0.0.jar
```

в папку:

```
plugins/
```

Итоговый путь должен выглядеть примерно так:

```
server/
└── plugins/
    └── OrynthMenus-1.0.0.jar
```

### 2. Установка FakeInventories

FakeInventories является обязательной зависимостью для всех инвентарных меню.

Поместите файл плагина в ту же папку:

```
plugins/FakeInventories-1.1.8.jar
```

> ⚠️ Без FakeInventories OrynthMenus не сможет открыть меню типа `inventory`.

### 3. Установка EconomyAPI

EconomyAPI требуется, если вы планируете использовать:

* магазины;
* скупщиков;
* проверку баланса;
* выдачу денег;
* списание денег;
* плейсхолдер баланса.

Файл также помещается в:

```
plugins/
```

Без EconomyAPI обычные меню и формы продолжат работать, но денежные действия будут недоступны.

### 4. Установка PlaceholderAPI-nukkit

PlaceholderAPI-nukkit не обязателен, но рекомендуется для серверов с большим количеством плагинов.

Он позволяет:

* использовать внешние плейсхолдеры в меню;
* передавать плейсхолдеры OrynthMenus другим плагинам;
* выводить статистику, ранги и другие динамические данные.

### 5. Первый запуск

После установки файлов полностью перезапустите сервер.

Не используйте только `/reload`, если плагин запускается впервые.

После запуска должна появиться папка:

```
plugins/OrynthMenus/
```

Пример структуры:

```
plugins/
└── OrynthMenus/
    ├── config.yml
    ├── permissions.yml
    ├── language/
    └── menus/
```

### 6. Проверка запуска

Проверьте консоль сервера.

При успешной загрузке не должно быть ошибок, связанных с:

```
OrynthMenus
FakeInventories
EconomyAPI
```

Также можно выполнить:

```
/orynthmenus reload
```

или короткий вариант:

```
/omenus reload
```

### Возможные проблемы

#### Меню не открывается

Проверьте:

* установлен ли FakeInventories;
* правильно ли указан ID меню;
* зарегистрирован ли файл в `config.yml`;
* нет ли ошибок YAML;
* существует ли команда открытия.

#### Деньги не работают

Убедитесь, что:

* EconomyAPI установлен;
* EconomyAPI успешно загрузился;
* в конфиге используется корректное действие;
* сумма указана числом.

#### Плагин не создаёт папку

Проверьте:

* совместимость версии NukkitMOT;
* права доступа к папке сервера;
* ошибки загрузки в консоли;
* наличие всех обязательных зависимостей.

### Рекомендуемый порядок установки

* [ ] Остановить сервер.
* [ ] Поместить OrynthMenus в `plugins`.
* [ ] Добавить FakeInventories.
* [ ] При необходимости добавить EconomyAPI.
* [ ] При необходимости добавить PlaceholderAPI-nukkit.
* [ ] Полностью запустить сервер.
* [ ] Проверить консоль.
* [ ] Убедиться, что появилась папка `plugins/OrynthMenus`.
* [ ] Выполнить `/orynthmenus reload`.
